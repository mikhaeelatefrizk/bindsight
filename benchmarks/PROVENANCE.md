# bindsight held-out evaluation set — provenance

This directory is the **held-out evaluation set** for bindsight: real,
clinically-pursued cell-surface antigens and **literature-validated binders**
against them, used to score how well a rediscovery run resurfaces known
targets (and to anchor binder-design benchmarks against real ground truth).

Every record is curated from canonical, globally-authoritative public
databases and carries a verifiable identifier. **No value is fabricated.**
The files are regenerated by [`build_eval_set.py`](build_eval_set.py), which
downloads binder sequences directly from the RCSB PDB so the ground-truth
sequences are byte-for-byte the deposited ones.

## Files

| File | What it is |
|---|---|
| `known.tsv` | Known antigens (targets) — symbol, UniProt, Ensembl gene, disease, tumor type, expected DE direction. Consumed by `bindsight benchmark --known-antigens`. |
| `binders.tsv` | Literature-validated binders — name, target, modality, clinical status, and citations (ChEMBL / NCT / PMID / DOI / PDB). |
| `binders.fasta` | Real VH/VL amino-acid sequences for the structurally-resolved binders, pulled from the cited PDB co-crystals. |
| `sources.json` | Machine-readable provenance: source databases, retrieval timestamp, and SHA-256 of each output. |
| `build_eval_set.py` | Reproducible generator (single source of truth). |

## Coverage

- **AML (the held-out set the user specified):**
  - **CD33** (`P20138`): gemtuzumab ozogamicin (approved), lintuzumab (Ph3),
    vadastuximab talirine (Ph2), Fab-10C8 (PDB `9VL2`, real VH/VL).
  - **CD123 / IL3RA** (`P26951`): tagraxofusp (approved), talacotuzumab/CSL362
    (Ph2/3), its murine parent 7G3 (PDB `4JZJ`, real VH/VL), flotetuzumab (Ph2).
- **Solid tumors (extends the benchmark):**
  - **HER2** (`P04626`): trastuzumab (PDB `1N8Z`), pertuzumab (PDB `1S78`).
  - **EGFR** (`P00533`): cetuximab (PDB `1YY9`).
  - **MSLN** (`Q13421`): amatuximab, anetumab ravtansine.
  - **CLDN6** (`P56747`): BNT211 (CLDN6 CAR-T).

## Data-quality note (correction made during curation)

The bindsight bundled accession map previously listed **`Q14953`** as CLDN6.
That is wrong: `Q14953` is **KIR2DS5** (killer-cell immunoglobulin-like
receptor 2DS5). The canonical CLDN6 accession is **`P56747`**
(`ENSG00000184697`), verified against UniProt. This eval set uses the correct
accession, and the bundled maps in the package are corrected to match.

## Honest scope

The binders here are **real, published binders used as ground truth** — not
outputs of the bindsight design pipeline. Sequences are provided only where a
public structure exists (`sequence_source = PDB:<id>`); agents whose sequences
are not in the public domain are marked `not_public` and carry their clinical
and database citations instead of an invented sequence.
